Frequently Asked Questions

Go to the Sky Note's homepage and click the button with 'Sign Up' text. Once clicked, you will be redirected to the Sign Up page. From there, you need to fill the fields present in the form to register. Once done, you will be redirected to the Sky Notes demo site.

Go to the Sky Note's homepage and click the button with 'Login' text. Once clicked, you will be redirected to the Login page. From there, you need to fill the fields present in the form. Once done, you will be redirected to the Sky Notes demo site.

In every page in the website except for the homepage, you will see either a moon image or a sun image in the top-right corner of the page. Once click, it will change the website's theme (from Dark Mode to Light Mode, or vice versa)

To specifically report a bug in the website, go to the navigation bar in the topmost side of the website (outside of the demo page). Click 'Contact', and from there, you can either send a message to the dev directly in the form provided, report an issue in the dev's GitHub repository by clicking the GitHub icon (1st row), message the dev directly in Gmail by clicking the Google icon (2nd row), or message the dev directly in Facebook by clicking the Facebook icon (3rd row).

Unfortunately, using this web application in phone devices (Android Phones and iPhones) are still unsupported. Hence, you cannot use the web application in phones yet.

Yes, your password and notes will be safely stored in this web application. We already implemented strict safety measures to keep your password from being known by others through the usage of PHP's BCrypt.

Unfortuantely, we are not accepting donations for this web application at the moment.
